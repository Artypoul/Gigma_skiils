# Capability Routing

| Requested outcome | Preferred destination | If unavailable |
| --- | --- | --- |
| Repository change or review | Project context/router and project-local workflow | Explain missing repository context or access. |
| Current or sourced answer | `research-with-evidence` plus a real source reader | State evidence gap. |
| Word, PDF, spreadsheet, or presentation | Matching document/artifact skill | Offer a conversational result or ask for an install only when useful. |
| Image or visual | Installed image/visual skill | Do not claim an image was created. |
| Connected service data/action | Connected app or MCP tool in scope | State the connector or authorization gap. |
| Durable background/recurring work | Real monitor, automation, or scheduler | Report that ongoing execution was not started. |

Never map a request directly to a provider-specific tool name copied from another prompt.
