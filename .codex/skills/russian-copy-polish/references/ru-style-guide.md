# Russian style guide for website copy

## Principles

- Сначала смысл, потом красота.
- Одно предложение — одна главная мысль.
- Глаголы сильнее отглагольных существительных.
- Конкретика сильнее абстракции.
- Уверенность сильнее крика.
- Доказательство сильнее обещания.

## Replace heavy constructions

| Weak | Better direction |
|---|---|
| осуществляем внедрение | внедряем |
| производим анализ | анализируем |
| является решением | помогает / решает / работает как |
| позволяет получить | даёт / помогает получить |
| с целью повышения | чтобы повысить |
| в рамках проекта | в проекте |
| на сегодняшний день | сейчас / убрать |

## Headings

Good headings should do at least one of these:

- explain the next section;
- name a user outcome;
- remove a doubt;
- show a difference;
- introduce proof.

Avoid headings that only decorate the page:

- «Наши преимущества»;
- «Почему мы»;
- «Инновационный подход»;
- «Комплексные решения».

## CTA and microcopy

- Use verbs.
- Name the next step.
- Avoid pressure without value.
- Explain response time or outcome when useful.

## Tone

For expert/B2B pages: calm, specific, confident.
For consumer/local services: clear, helpful, practical.
For startup/SaaS: direct, product-led, not overhyped.

## Final checks

- Can the sentence be shorter?
- Does this phrase appear on thousands of other sites?
- Is the claim supported?
- Would a real customer say it this way?
- Did we accidentally change the meaning?
