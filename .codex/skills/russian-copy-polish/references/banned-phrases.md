# Banned and weak phrases

These phrases are not forbidden in every possible context, but they usually signal generic copy. Replace them with concrete meaning.

## Generic marketing phrases

| Weak phrase | Replace with |
|---|---|
| комплексные решения | scope, deliverables, modules, process |
| индивидуальный подход | what is actually customized |
| инновационные технологии | technology name and why it matters |
| широкий спектр услуг | list of relevant services or packages |
| команда профессионалов | expertise, roles, cases, credentials |
| высокое качество | measurable quality criteria |
| доступные цены | pricing principle or value explanation |
| быстрые сроки | actual timeline or process |
| надёжный партнёр | proof, retention, SLA, process |
| выведите бизнес на новый уровень | specific business outcome |
| в современном мире | remove |
| динамично развивающийся | remove or prove with facts |
| полный цикл | define stages and ownership |
| под ключ | define what is included and excluded |

## AI-smell phrases

- «важно отметить»;
- «стоит подчеркнуть»;
- «это не просто X, а Y»;
- «в эпоху цифровизации»;
- «уникальное сочетание»;
- «максимально эффективный»;
- «лучшие практики» without specifics.

## Safer replacement pattern

```text
Weak: Мы предлагаем комплексные решения для бизнеса.
Better: Проектируем структуру сайта, SEO-страницы и тексты так, чтобы команда могла сразу передать их дизайнеру и разработчику.
```

## Rule

Do not only delete weak phrases. Replace them with one of:

- audience;
- outcome;
- mechanism;
- proof;
- scope;
- next step;
- limitation.
