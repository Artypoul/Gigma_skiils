# Диагностика подключения «Твой ВПС» + эскалация

Цель — починить силами пользователя, а если не выходит или дело в сервисе — передать Артёму с готовыми данными. Веди по шагам, по одному действию, дожидаясь результата.

Формулировки — нейтральные (см. [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md): «приложение», «подключение к сервису», «подписка»; без «обхода/разблокировки»). Факты и канал связи — там же.

## Сначала — что именно не так?

Уточни одним вопросом, выбрав вариант:

1. **Не могу войти в кабинет.**
2. **Приложение не подключается.**
3. **Подключилось, но сайты не открываются.**
4. **Подключено, но очень медленно.**
5. **Пишет про оплату или что срок истёк.**
6. **В кабинете нет ссылки-подписки (пусто).**

Дальше — соответствующий раздел ниже.

## 1. Не могу войти в кабинет

Вход — по номеру телефона, звонком, без пароля (подробно — скил **vps-connect**, Шаг 0 и 0.5):
- Проверить формат номера: `+7`, 11 цифр.
- Истёк таймер ожидания звонка → начать заново («Отменить и ввести другой номер»).
- Звонок не проходит → попробовать позже; не выходит → **эскалация** (с номером телефона).

## 2. Приложение не подключается

1. **Сменить сервер/конфиг** в списке подписки — выбрать другой и подключиться (бывает, конкретный конфиг не идёт, а соседний работает).
2. **Обновить подписку** в приложении (на телефоне — потянуть вниз; на компьютере — кнопка «Обновить»).
3. **Переустановить профиль:** удалить подписку в приложении и добавить заново по ссылке из кабинета (скил **vps-connect**).
4. **Проверить обычный интернет** — открывается ли сайт при выключенном подключении. Нет интернета вообще → проблема не в нас.
5. **Обновить приложение:** открыть App Store (iPhone) или Google Play (Android) → найти **Happ** → если есть кнопка «Обновить», нажать (неактивна = версия уже свежая).

## 3. Подключается, но сайты не открываются

1. **Сменить конфиг/сервер** из подписки.
2. Проверить **дату и время** на устройстве (должны быть точные/авто) — сбитое время ломает соединение.
3. Отключить **другие подобные приложения/прокси**, антивирус с «защитой сети», если включены.
4. Переустановить профиль (как в п.2.3).

## 4. Работает, но медленно

1. Попробовать **другой конфиг** из подписки.
2. Сравнить скорость **с выключенным подключением**.
3. Вечерние часы пик — возможны просадки.
4. Если **на всех конфигах стабильно медленно** — это сторона сервиса → **эскалация**.

## 5. Истёк срок / лимит / просит оплату

Свериться со статусом подписки в кабинете (карта фактов):
- `expired` (закончился) → продлить в `/select-plan` → `/buy`. **Аккаунт и настройки остаются на месте**, доступ восстановится после оплаты.
- `limited` (лимит трафика на тарифе) → сменить тариф / дождаться сброса.
- `disabled` «Ожидая оплату» → оплата не прошла, повторить оплату.
- Вопрос про **возврат** → форма `/support`, тема «Возврат» (доступна для активной платной подписки).

## 6. Пустая ссылка-подписка / её нет в кабинете

1. Обновить страницу `/home` — для нового пользователя ссылка появляется не мгновенно, подождать несколько секунд.
2. Проверить статус подписки на `/home` (если не активна — см. раздел 5).
3. Перезайти в кабинет.
4. Если так и пусто → **эскалация**.

## Когда эскалировать Артёму

Эскалируй, если: шаги не помогли; ИЛИ признаки стороны сервиса (медленно на всех конфигах, пустая подписка после перезахода, статус не сходится с оплатой); ИЛИ деньги/возврат не решаются в кабинете.

**Куда:** Telegram-канал Артёма (адрес — [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md), раздел «Связь и эскалация»); запасной путь — форма `/support` в кабинете.

**Собрать перед отправкой (6 пунктов):**
1. Устройство и ОС.
2. Приложение и версия (например HAPP 2.x).
3. Номер телефона (логин в кабинете) + тариф и статус.
4. Что не работает и с какого момента.
5. Что уже пробовали (из шагов выше).
6. **Скриншот ошибки.** Как сделать: iPhone — кнопки громкость + боковая; Android — громкость вниз + питание; Windows — `Win+Shift+S` или `Print Screen`; macOS — `Cmd+Shift+4`. Затем прикрепить скриншот в Telegram или в форму `/support`.

⚠ Пароль не спрашивать. Ссылку-подписку не публиковать в открытых чатах. Не обсуждать обход блокировок/доступ к запрещённому — оставаться в рамках «подписка + приложение».
