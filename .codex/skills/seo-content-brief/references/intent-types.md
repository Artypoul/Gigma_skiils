# Intent types

Use intent to decide page shape. Do not force every query into the same article template.

| Intent | User wants | Best page types | CTA |
|---|---|---|---|
| Informational | understand a topic or solve a problem | guide, how-to, checklist, explainer | related guide, template, newsletter, soft product link |
| Commercial investigation | compare options before buying | comparison, alternatives, buyer guide, use-case page | demo, trial, consultation, pricing |
| Transactional | buy, book, download, start | landing page, pricing, product/service page | start, buy, request quote, book call |
| Navigational | reach a known brand/page | brand page, support/docs page | direct path to requested action |
| Local | find a provider in a location | local service page, contact page, location page | call, book, get directions, request quote |
| Implementation | make something work after choosing | docs, tutorial, template, onboarding guide | setup, integrate, contact support |
| Comparison | decide between A and B | compare page, alternative page, table, decision guide | see why switch, demo, pricing |

## Mixed intent warning

A query may have mixed intent. Choose the dominant intent and support secondary intent without bloating the page.

Example:

- «seo audit checklist» → informational + template download.
- «best crm for agencies» → commercial investigation.
- «crm pricing» → transactional/commercial.

## Structure by intent

### Informational

1. Quick answer.
2. Definition/context.
3. Steps/framework.
4. Examples.
5. Common mistakes.
6. Template/tool.
7. Related next step.

### Commercial

1. Who this is for.
2. Decision criteria.
3. Options/alternatives.
4. Comparison table.
5. Proof.
6. Recommendation by use case.
7. CTA.

### Transactional

1. Clear offer.
2. Benefits/outcomes.
3. How it works.
4. Proof/risk reducers.
5. Pricing/scope if relevant.
6. FAQ.
7. CTA.
