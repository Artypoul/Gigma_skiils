# Подключение к «Твой ВПС»

Цель — довести человека до подключения. Общий путь: **войти в кабинет → скопировать ссылку-подписку → поставить приложение и добавить в него подписку → подключиться**. Объясняй по одному шагу.

Формулировки — нейтральные (см. [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md)): «приложение», «подключение к сервису», «подписка». Без «обхода» и «разблокировки».

## Шаг 0. Войти в кабинет и взять ссылку-подписку

Вход в кабинет «Твой ВПС» — **по номеру телефона, без пароля** («вход звонком»):

1. Открыть страницу входа, ввести свой номер телефона (`+7…`) и отметить согласие.
2. Нажать **«Войти звонком»**.
3. На экране появится номер — **позвонить на него со своего телефона** (звонок бесплатный, его автоматически сбросят).
4. Вход подтвердится сам, кабинет откроется. **Код вводить не нужно.**

Дальше на **главной** странице (`/home`) найти **ссылку-подписку** и нажать «Скопировать».

> Ссылка личная — не пересылать в открытые чаты. Не получается войти → **Шаг 0.5**. Ссылки нет / пусто → скил **vps-troubleshoot**.

## Шаг 0.5. Если не получается войти

- **Номер не принимается** → проверить формат: российский, начинается с `+7`, всего 11 цифр.
- **Не дождались / истёк таймер** → нажать «Отменить и ввести другой номер» и начать заново.
- **Звонок не проходит** (нет связи, номер заблокирован у оператора) → попробовать чуть позже; если не выходит — написать в поддержку (Telegram **@artypoul** или форма `/support`) с указанием своего номера телефона.

## Шаг 1. Выбрать приложение

- **HAPP — рекомендуется.** Подробная пошаговая настройка (установка, добавление подписки, подключение, обновление) — в скиле **happ-setup**. Туда и веди для HAPP.
- **Другие приложения** (если человек уже ими пользуется) — та же ссылка-подписка:

| Платформа | Приложение | Как добавить |
|---|---|---|
| Android | v2rayNG, Hiddify, NekoBox | «+» → «Импорт из буфера обмена» → «Обновить подписку» → выбрать сервер → подключиться |
| iPhone/iPad | Streisand, V2Box, Shadowrocket | «+» / «Добавить подписку» → вставить ссылку → обновить → выбрать сервер → подключиться |
| Windows / macOS | Hiddify, NekoRay | «Новый профиль» / «Добавить из буфера» → вставить ссылку → выбрать сервер → подключиться |

Во всех приложениях схема одинаковая: **вставить ссылку-подписку → обновить список → выбрать сервер → подключиться**.

## Проверка, что всё работает

1. После подключения в приложении рядом с сервером/кнопкой видно **«подключено»**.
2. Открыть в браузере любой обычный сайт — он открывается.

Не подключилось или сайты не открываются → скил **vps-troubleshoot**.
