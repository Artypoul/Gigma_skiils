# Установка и настройка приложения для «Твой ВПС»

Цель — помочь человеку скачать приложение и подключиться к своему доступу. Принцип один для всех приложений: **скопировать ссылку-подписку из кабинета → добавить её в приложение → выбрать сервер → подключиться**. Объясняй по одному шагу, дожидаясь результата.

Формулировки — нейтральные (см. [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md), раздел «Формулировки и правовая рамка»): «приложение», «подключение к сервису», «подписка». Без «обхода» и «разблокировки».

## Шаг 0. Взять ссылку-подписку

1. Войти в личный кабинет «Твой ВПС» (вход по email + код из письма).
2. На **главной** странице (`/home`) найти **ссылку-подписку** и нажать «Скопировать».

> Это личная ссылка — не пересылать в открытые чаты. Если её нет/пусто — см. скил **vps-troubleshoot** (возможно, подписка не активна).

## HAPP — пошагово (основное приложение)

### Шаг 1. Установить HAPP

Скачать с официального сайта **https://happ.info** (бывший happ.su — теперь ведёт сюда). Версии:
- **iPhone/iPad (iOS)** — App Store. ⚠ Есть **отдельная российская версия** — если глобальная в вашем App Store не открывается, ставьте российскую.
- **Android** — Google Play или файл APK с сайта.
- **Windows / macOS / Linux** — установщик с сайта (у macOS тоже есть глобальная и российская версии).

### Шаг 2. Добавить подписку (3 способа — выбрать любой)

Сначала скопировать ссылку-подписку из кабинета (Шаг 0). Дальше любой из способов:

1. **Одной кнопкой (самое простое).** Если в кабинете есть кнопка **«Добавить в Happ»** или **QR-код** — нажать кнопку (или отсканировать QR). Happ откроется и добавит подписку сам.
2. **Сканировать QR.** В Happ открыть добавление → «Сканировать QR» → навести на код из кабинета.
3. **Вставить ссылку вручную.** Открыть Happ → **«+»** / «Добавить подписку» → **«Вставить из буфера обмена»** (ссылка уже скопирована).

После добавления Happ загрузит список серверов.

### Шаг 3. Подключиться

1. Выбрать сервер из списка (если несколько — начать с верхнего).
2. Нажать **большую круглую кнопку** включения (по центру).
3. **В первый раз** телефон попросит разрешить настройку профиля подключения (в системном окне он называется «VPN-профиль») — нажать **«Разрешить»**. Запрос появляется один раз.
4. Готово — кнопка станет активной, рядом будет «подключено».

### Шаг 4. Обновить подписку (когда нужно)

Если поменялся набор серверов или после продления тарифа — в Happ потянуть список **вниз** (обновить) или в подписке нажать **«Обновить»**. Свежие серверы подтянутся.

> Иконки и названия пунктов немного отличаются между iPhone / Android / компьютером и версиями приложения. **Суть одна:** добавить подписку (кнопкой / QR / вставкой) → выбрать сервер → нажать подключение → один раз разрешить профиль.

## Другие приложения (та же ссылка-подписка)

| Платформа | Приложение | Как добавить |
|---|---|---|
| Android | **v2rayNG**, Hiddify, NekoBox | «+» → «Импорт из буфера обмена» (Import config from Clipboard) → затем «Обновить подписку» → выбрать сервер → подключиться |
| iPhone/iPad | **Streisand**, V2Box, Shadowrocket, HAPP | «+» / «Добавить подписку» → вставить ссылку → обновить → выбрать сервер → подключиться |
| Windows / macOS | **Hiddify**, HAPP, NekoRay | «Новый профиль» / «Добавить из буфера» → вставить ссылку подписки → выбрать сервер → подключиться |

Во всех приложениях схема одинаковая: **вставить ссылку-подписку → обновить список → выбрать сервер → подключиться**.

## Проверка, что всё работает

1. После подключения в приложении рядом с сервером/кнопкой видно состояние **«подключено»**.
2. Открыть в браузере любой обычный сайт — он открывается.

Не подключилось или сайты не открываются после подключения → скил **vps-troubleshoot**.
