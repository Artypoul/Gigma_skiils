# Установка и настройка приложения для «Твой ВПС»

Цель — довести человека до подключения. Принцип один для всех приложений: **войти в кабинет → скопировать ссылку-подписку → добавить её в приложение → выбрать сервер → подключиться**. Объясняй по одному шагу, дожидаясь результата.

Формулировки — нейтральные (см. [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md), раздел «Формулировки и правовая рамка»): «приложение», «подключение к сервису», «подписка». Без «обхода» и «разблокировки».

## Шаг 0. Войти в кабинет и взять ссылку-подписку

Вход в кабинет «Твой ВПС» — **по номеру телефона, без пароля** («вход звонком»):

1. Открыть страницу входа, ввести свой номер телефона (`+7…`) и отметить согласие.
2. Нажать **«Войти звонком»**.
3. На экране появится номер — **позвонить на него со своего телефона** (звонок бесплатный, его автоматически сбросят).
4. Вход подтвердится сам, кабинет откроется. **Код вводить не нужно.**

Дальше на **главной** странице (`/home`) найти **ссылку-подписку** и нажать «Скопировать».

> Ссылка личная — не пересылать в открытые чаты. Не получается войти → **Шаг 0.5**. Ссылки нет / пусто → скил **vps-troubleshoot**.

## Шаг 0.5. Если не получается войти

- **Номер не принимается** → проверить формат: российский, начинается с `+7`, всего 11 цифр.
- **Не дождались / истёк таймер** → нажать «Отменить и ввести другой номер» и начать заново.
- **Звонок не проходит** (нет связи, номер заблокирован у оператора) → попробовать чуть позже; если не выходит — написать в поддержку (Telegram **@artypoul** или форма `/support`) с указанием своего номера телефона.

## Шаг 1. Установить HAPP

Скачать с официального сайта **https://happ.info** (бывший happ.su — теперь ведёт сюда). Версии:
- **iPhone/iPad (iOS)** — App Store, искать по слову **Happ**. ⚠ Есть **отдельная российская версия** — если глобальная в вашем App Store не открывается, ставьте российскую. Если **ни одна** не ставится (региональное ограничение) — написать в поддержку, подскажем альтернативу (Streisand / V2Box).
- **Android** — Google Play или файл APK с сайта.
- **Windows / macOS / Linux** — установщик с сайта (у macOS тоже есть глобальная и российская версии).

## Шаг 2. Добавить подписку (выбрать самый простой доступный способ)

Сначала скопировать ссылку-подписку (Шаг 0). Затем — **от лёгкого к сложному**, первый доступный:

1. **Кнопкой (лучший способ).** Если в кабинете есть кнопка **«Добавить в Happ»** — нажать её. Happ откроется и добавит подписку сам.
2. **QR-кодом.** Если есть QR-код — в Happ открыть добавление → «Сканировать QR» → навести на код из кабинета.
3. **Вставить ссылку вручную.** Открыть Happ → **«+»** / «Добавить подписку» → **«Вставить из буфера обмена»** (ссылка уже скопирована).

После добавления Happ загрузит список серверов.

## Шаг 3. Подключиться

1. Выбрать сервер из списка (если несколько — начать с верхнего).
2. Нажать **большую круглую кнопку** включения (по центру экрана).
3. **В первый раз** телефон/компьютер попросит разрешить настройку профиля подключения (в системном окне он называется «VPN-профиль») — нажать **«Разрешить»**. Запрос появляется один раз.
4. Готово — кнопка станет активной, рядом будет «подключено».

## Шаг 4. Обновить подписку (когда нужно)

Если поменялся набор серверов или после продления тарифа — обновить подписку:
- **на телефоне** — потянуть список вниз;
- **на компьютере** — нажать кнопку **«Обновить»** у подписки.

Свежие серверы подтянутся.

## Проверка, что всё работает

1. После подключения в приложении рядом с сервером/кнопкой видно **«подключено»**.
2. Открыть в браузере любой обычный сайт — он открывается.

Не подключилось или сайты не открываются → скил **vps-troubleshoot**.

> **Если человек застрял** на каком-то шаге — попроси сказать, **на каком именно шаге и что на экране** (можно скриншот), и веди точечно. Если кнопки в приложении называются чуть иначе, чем тут, — ориентируйся на смысл шага, а не на точную подпись (версии отличаются).
