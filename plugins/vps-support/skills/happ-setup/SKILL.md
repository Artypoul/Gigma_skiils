# Настройка приложения HAPP

Пошаговая настройка HAPP для подключения к сервису «Твой ВПС». Сверено с официальной инструкцией Happ (ссылки внизу). Веди по одному шагу, дожидаясь результата; говори простым языком.

Нужна **ссылка-подписка** из кабинета — как её взять и как войти, см. скил **vps-connect** (Шаг 0). Формулировки нейтральные (см. [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md)): «приложение», «подписка», «подключение». Без «обхода/разблокировки».

## Шаг 1. Установить HAPP

Скачать с официального сайта **https://happ.info** (бывший happ.su):
- **iPhone/iPad (iOS)** — App Store, искать **Happ**. ⚠ Есть **отдельная российская версия** — если глобальная в вашем App Store не открывается, ставьте российскую. Если **ни одна** не ставится (региональное ограничение) — написать в поддержку, подскажем альтернативу (Streisand / V2Box).
- **Android** — Google Play или APK с сайта.
- **Windows / macOS / Linux** — установщик с сайта (у macOS тоже есть глобальная и российская версии).
- Есть версии и для **Android TV / Apple TV** (отдельные гайды в FAQ Happ).

## Шаг 2. Добавить подписку

Сначала скопировать ссылку-подписку из кабинета. Затем — любой из **трёх официальных способов** (от простого к сложному):

1. **Кнопкой / deep link (проще всего).** Если в кабинете есть кнопка **«Добавить в Happ»** — нажать её: Happ откроется и добавит подписку сам.
2. **QR-кодом.** В Happ нажать **«+»** (в правом верхнем углу) → выбрать сканирование QR → навести на код из кабинета.
3. **Из буфера обмена.** Скопировать ссылку → в Happ нажать **«+»** (правый верхний угол) → «Добавить подписку» / «Вставить из буфера».

После добавления Happ загрузит список серверов.

> По официальной инструкции Happ поддерживает обычную ссылку-подписку и зашифрованную (`happ://crypto…`) — обе добавляются одинаково, приложение разберётся само. Пункт **«Manual Input»** (ручной ввод протокола и полей) для нашего сервиса **не нужен** — у нас всё в одной ссылке-подписке.

## Шаг 3. Подключиться

1. Выбрать сервер из списка (если несколько — начать с верхнего).
2. Нажать **большую круглую кнопку** включения (по центру).
3. **В первый раз** устройство попросит разрешить настройку профиля подключения (в системном окне он называется «VPN-профиль») — нажать **«Разрешить»**. Появляется один раз.
4. Готово — рядом с сервером/кнопкой появится «подключено».

## Шаг 4. Обновить подписку

Когда поменялся набор серверов или после продления тарифа:
- **на телефоне** — потянуть список подписки **вниз**;
- **на компьютере** — нажать кнопку **«Обновить»** у подписки.

## Если не получается

- Конкретный сервер не подключается → выбрать **другой сервер** из списка.
- Не помогло / другая проблема → скил **vps-troubleshoot** (диагностика по шагам и эскалация).

> Иконки и подписи кнопок немного отличаются между iPhone / Android / компьютером и версиями Happ. Суть одна: **добавить подписку (кнопка/QR/буфер) → выбрать сервер → подключиться → один раз разрешить профиль.** Если человек застрял — попроси сказать, на каком шаге и что на экране.
