# Установка и настройка приложения для «Твой ВПС»

Цель — помочь человеку скачать приложение и подключиться к своему доступу. Принцип один для всех приложений: **скопировать ссылку-подписку из кабинета → добавить её в приложение → выбрать сервер → подключиться**. Объясняй по одному шагу, дожидаясь результата.

Формулировки — нейтральные (см. [`reference/vps-service-facts.md`](../../reference/vps-service-facts.md), раздел «Формулировки и правовая рамка»): «приложение», «подключение к сервису», «подписка». Без «обхода» и «разблокировки».

## Шаг 0. Взять ссылку-подписку

1. Войти в личный кабинет «Твой ВПС» (вход по email + код из письма).
2. На **главной** странице (`/home`) найти **ссылку-подписку** и нажать «Скопировать».

> Это личная ссылка — не пересылать в открытые чаты. Если её нет/пусто — см. скил **vps-troubleshoot** (возможно, подписка не активна).

## HAPP — основное приложение

Установка: App Store / Google Play / сайт **https://happ.su** (есть версии для iPhone, Android, Windows, macOS, Android TV).

Подключение:
1. Скопировать ссылку-подписку (Шаг 0).
2. Открыть **HAPP**.
3. Нажать **«+»** (добавить) → выбрать **«Добавить подписку»** / «Вставить из буфера обмена» — HAPP подхватит ссылку и загрузит список серверов.
   - Если в кабинете есть кнопка/QR **«Добавить в HAPP»** — можно нажать её: приложение откроется и добавит подписку само.
4. Дождаться, пока подтянется список серверов.
5. Выбрать сервер и нажать **большую кнопку подключения** (по центру).
6. Разрешить системе настройку соединения (запрос появится один раз).
7. Готово — рядом с сервером/кнопкой появится «подключено».

> Названия пунктов могут чуть отличаться по версии приложения — суть одна: «добавить подписку из буфера → выбрать сервер → подключиться».

## Другие приложения (та же ссылка-подписка)

| Платформа | Приложение | Как добавить |
|---|---|---|
| Android | **v2rayNG**, Hiddify, NekoBox | «+» → «Импорт из буфера обмена» (Import config from Clipboard) → затем «Обновить подписку» → выбрать сервер → подключиться |
| iPhone/iPad | **Streisand**, V2Box, Shadowrocket, HAPP | «+» / «Добавить подписку» → вставить ссылку → обновить → выбрать сервер → подключиться |
| Windows / macOS | **Hiddify**, HAPP, NekoRay | «Новый профиль» / «Добавить из буфера» → вставить ссылку подписки → выбрать сервер → подключиться |

Во всех приложениях схема одинаковая: **вставить ссылку-подписку → обновить список → выбрать сервер → подключиться**.

## Проверка, что всё работает

1. После подключения в приложении рядом с сервером/кнопкой видно состояние **«подключено»**.
2. Открыть в браузере любой обычный сайт — он открывается.

Не подключилось или сайты не открываются после подключения → скил **vps-troubleshoot**.
